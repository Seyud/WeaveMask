# WeaveMask 更新日志

### v30.7.5

- 将 UI 从 Views/XML/Fragments 完全重写为 Jetpack Compose + Miuix 组件库
- 将导航系统从 Fragment Navigation Component + SafeArgs 迁移至 Navigation3，支持类型安全的自定义 `Navigator` 和弹簧物理过渡动画
- 新增 6 种主题颜色模式：跟随系统、浅色、深色、Monet 跟随系统、Monet 浅色、Monet 深色
- 新增 16 种 Monet 主题预设主色调（红、粉、紫、深紫、靛、蓝、青、碧、绿、黄、琥珀、橙、棕、蓝灰、樱花、棉花糖）
- 新增双主页布局：经典和 Weavsk，可在设置中切换
- 重新设计关于页面，支持动态着色器背景（`BgEffectBackground`）、视差滚动动画和 Miuix 模糊混合效果
- 新增 Liquid Glass 视觉组件：`CombinedBackdrop`、`InnerShadow`、`Lens`、`Vibrancy`
- 新增浮动底栏，支持 Liquid Glass 模糊效果开关（API 33+）
- 新增模糊效果开关（API 31+）
- 新增页面缩放滑块（80%–110%），支持吸附点
- 新增独立的应用语言选择页面
- 新增应用图标切换器：当前图标、Legacy Weave、Legacy Mask 三种变体，通过 `AppIconManager` 管理
- 新增 Android 14+ 预测性返回手势开关
- 新增白名单/黑名单超级用户模式，对接同步 Zygisk Next 白名单模式
- 新增智能模式开关，作为超级用户列表模式选择器的主开关
- 白名单模式下通过 `WhitelistModePackageSyncReceiver` 自动同步新安装应用到 DenyList
- 新增白名单/黑名单模式切换时的 DenyList 快照/恢复功能
- 新增策略备份/恢复，支持 FBE 设备持久化
- 超级用户列表中显示所有已安装应用（不仅限于已有 root 策略的应用）
- 白名单模式下长按应用图标显示操作菜单
- 新增模块仓库浏览器，兼容 KernelSU 模块仓库格式
- 新增模块操作和模块 WebUI 的主屏幕快捷方式，支持自定义图标和 `root://` 协议
- 支持批量本地模块安装（一次安装多个 ZIP 文件）
- 新增可从模块列表访问的独立模块操作页面
- 集成 WebUI 主题注入（`MonetColorsProvider`）— 模块 WebUI 跟随应用的 Monet 配色方案
- 新增下载并修补安装方式，支持下载启动镜像并修补
- 支持通过 Intent 过滤器从外部应用打开 `.zip` 文件
- 更名为 WeaveMask，使用自定义包名 `io.github.seyud.weave`
- 将所有 Rikka UI 依赖替换为 Miuix 组件
- 将 KAPT 注解处理替换为 Kotlin Serialization 和 Compose 编译器插件
- 将 `buildSrc` 迁移至 `build-logic` 组合构建
- 将 stub 资源拆分为独立的 `stub-res` 模块
- 修复 RTL 语言（阿拉伯语）在设置页面的滚动卡顿
- 修复通过 `enableEdgeToEdge` 导致的闪屏页面紫色背景泄漏
- 修复 stub manifest 中 `excludeFromRecents` 导致隐藏模式下应用从最近任务列表消失
- 修复模块仓库详情页退出时 RenderThread SIGSEGV 崩溃
- 修复 Monet 主题闪屏页面未应用自定义种子颜色
- 修复主题切换后导航回主页的问题
- 修复模块更新按钮刷新闪烁
- 修复标签切换页面闪烁
- 修复 stub 在随机包名 APK 上本地模块 WebUI 的 `ERR_CLEARTEXT_NOT_PERMITTED` 错误
- 修复白名单模式在 FBE 设备上的持久化问题和模块页面不可点击状态
- 修复白名单模式残留导致已授权应用无法获取 root 及重启后策略丢失
- 修复 Windows ONDK 运行时找不到 cargo DLL
- 修复 `su_request_dialog` 布局问题
